Overview

NgPlusPlus is an MCP (Model Context Protocol) server that helps teams upgrade Angular applications safely and efficiently. It automates the most error-prone parts of a major Angular upgrade — resolving transitive dependency conflicts, validating version compatibility, and suggesting migration paths — directly inside any MCP-compatible AI assistant.

Instead of spending hours manually cross-referencing changelogs and peer-dependency matrices, NgPlusPlus exposes a set of smart tools that your AI agent can call to handle the heavy lifting.

Features

🔍

Intelligent Dependency Resolution

Automatically resolves transitive dependency conflicts while preserving compatibility across the entire dependency tree.

📦

Version Management

Wraps npm-check-updates with additional conflict detection, doctor mode, and semver validation.

✅

Compatibility Checking

Validates peer dependencies and engine constraints before any changes land in your package.json.

🤖

AI-Powered Suggestions

Uses OpenAI to rank candidate package versions and generate upgrade strategies when purely algorithmic resolution hits its limits.

🔌

MCP Integration

Works out-of-the-box with any MCP-compatible client — Claude Desktop, VS Code Copilot, and more.

📋

Structured Logging

File-based logging with rotation and configurable levels so every resolution decision is fully traceable.

Available Tools

Dependency Resolver

dependency_resolver_update

Updates packages to target versions and automatically resolves any transitive dependency conflicts introduced by the upgrade.

  • Updates specified packages to requested target versions
  • Detects and resolves upstream transitive dependency conflicts
  • Returns a detailed changelog of every version that was touched

NCU Update

ncu_update

Enhanced wrapper around npm-check-updates for checking and applying the latest compatible versions.

  • Scan for newer package versions without modifying anything
  • Apply updates directly to package.json
  • Optional doctor mode to detect breaking changes

Dumb Resolver

dumb_resolver

A lightweight, deterministic conflict resolver that operates without AI — useful for offline scenarios or simple upgrades.

Echo Tools

echo  multi_echo  echo_formatted  echo_ai_enhanced

Utility tools for testing the MCP connection and demonstrating AI enhancement capabilities.

  • echo — basic round-trip text echo
  • multi_echo — echo multiple strings with formatting options
  • echo_formatted — uppercase, lowercase, title-case transforms
  • echo_ai_enhanced — grammar correction, style improvements, translation via OpenAI

Quick Start

# 1. Clone & install
git clone https://github.com/arklnd/NgPlusPlus.git
cd NgPlusPlus
npm install

# 2. Build
npm run build

# 3. Add to your MCP client config (e.g., Claude Desktop)
# {
#   "mcpServers": {
#     "ngplusplus": {
#       "command": "node",
#       "args": ["path/to/NgPlusPlus/dist/index.js"]
#     }
#   }
# }

# 4. Run tests
npm test
